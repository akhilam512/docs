## Overview

The Pipecat client SDKs provide a variety of transport packages for real-time communication with a Voice-to-Voice LLM servers. The RealTimeWebSocketTransport provides an abstraction for these types of transports to support direct connections to the LLM server and stream media.

<Note>
Transports of this type are primarily intended for dev-only and testing purposes. Since this transport talks directly with the LLM, there is no way to obscure API Keys necessary for doing so. Developers will need to eventually build a server component with a server-friendly transport, like the [DailyTransport](./daily).
</Note>
